fun <reified K, reified V> MutableMap<*, *>.asMutableMapOfType(): MutableMap<K, V>?

Cast MutableMap with raw type to specified K and V types.

Checks is this MutableMap contains elements of specified K and V types. If this is true then receiver are casted to MutableMap with K and V types.

Receiver
MutableMap with raw types.

Return
casted current MutableMap with raw type to MutableMap with K and V types, null otherwise.
