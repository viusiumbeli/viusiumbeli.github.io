fun <reified T> MutableList<*>.asMutableListOfType(): MutableList<T>?

Cast MutableList with raw parameter type to specified parameter T.

Checks is this MutableList contains elements of specified parameter T type. If this is true then receiver are casted to MutableList with specified T type.

Receiver
MutableList with raw type.

Return
casted current MutableList with raw type to MutableList with T type, null otherwise.
